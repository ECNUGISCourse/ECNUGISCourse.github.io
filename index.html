Numpy、SciPy、MatPlotLib在Python2.7.9下的安装与配置

Posted on 2017-03-03

1.简单粗暴，使用可执行文件直接安装

在安装前首先需要配置好环境变量，具体方法参见“ArcGIS下的python环境变量配置”一文
配置完成后，下载这些模块的exe可执行文件
目前，numpy的最新版本为1.12.0，打包的可执行文件最新版本为1.10.1，下载地址numpy-1.10.1-win32-superpack-python2.7.exe
下载成功后，双击即可安装到对应python路径

2. pip安装

可以直接运行pip install numpy安装。
也可以在python包管理网站下载numpy的whl文件包，使用pip install numpy-1.12.0-cp27-none-win32.whl运行安装

注意，安装可能出现No module name six的错误，可以参照pip管理包的方法安装six模块

ArcGIS下的python环境变量配置

作为GIS学生，我们的计算机上一定安装了ArcGIS系列软件，而ArcGIS软件依赖内置的python解释器实现了Arcpy功能。因此我们需要对ArcGIS自带的python解释器进行适当配置以便流畅使用。这里介绍如何配置python解释器的环境变量。

python安装目录下的文件结构大致如下：

如下图打开“环境变量”设置窗口

找到名为Path的系统变量，如果没有，则新建一个。

双击此项，进行编辑

在编辑窗口中新建变量的值，并输入以下文件夹的绝对路径：

  • python解释器的安装目录，如：C:\Python27\ArcGIS10.3
  • Scripts文件夹，如：C:\Python27\ArcGIS10.3\Scripts

安装和使用pip管理Python第三方模块

Posted on 2017-02-15

pip是python社区提供的包管理器，使用它可以方便地安装、卸载和更新python的第三方包。

使用脚本安装和升级pip

要安装或升级pip，需要下载get-pip.py。
然后运行以下命令（需要管理员权限）
python get-pip.py
如果setuptools(或distribute)未安装, get-pip.py会自动为你安装setuptools

如果需要升级setuptools(或distribute), 运行pip install -U setuptools

安装包（package）

pip支持从 PyPI、版本控制系统、本地项目, 甚至直接通过distribution文件安装package.
最常见的是使用Requirement Specifiers从 PyPI 安装。
pip install SomePackage # 最新版本
pip install SomePackage==1.0.4 # 指定版本
pip install ‘SomePackage>=1.0.4’ # 最低版本
可以使用Requirements文件列出需要安装的packages，并通过pip install来安装：
pip install -r requirements.txt
这个文件的格式详情可以参考：Requirements File Format
还可以通过Wheels文件安装，即：
pip install SomePackage-1.0-py2.py3-none-any.whl
Wheel文件可以在PyPI网页上下载

卸载包

pip uninstall SomePackage

更新包

pip install --upgrade SomePackage

查看和列举包

pip list

查找包

pip search "query statement"
