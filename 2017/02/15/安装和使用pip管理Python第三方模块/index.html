安装和使用pip管理Python第三方模块

Posted on 2017-02-15 |

pip是python社区提供的包管理器，使用它可以方便地安装、卸载和更新python的第三方包。

使用脚本安装和升级pip

要安装或升级pip，需要下载get-pip.py。
然后运行以下命令（需要管理员权限）
python get-pip.py
如果setuptools(或distribute)未安装, get-pip.py会自动为你安装setuptools

如果需要升级setuptools(或distribute), 运行pip install -U setuptools

安装包（package）

pip支持从 PyPI、版本控制系统、本地项目, 甚至直接通过distribution文件安装package.
最常见的是使用Requirement Specifiers从 PyPI 安装。
pip install SomePackage # 最新版本
pip install SomePackage==1.0.4 # 指定版本
pip install ‘SomePackage>=1.0.4’ # 最低版本
可以使用Requirements文件列出需要安装的packages，并通过pip install来安装：
pip install -r requirements.txt
这个文件的格式详情可以参考：Requirements File Format
还可以通过Wheels文件安装，即：
pip install SomePackage-1.0-py2.py3-none-any.whl
Wheel文件可以在PyPI网页上下载

卸载包

pip uninstall SomePackage

更新包

pip install --upgrade SomePackage

查看和列举包

pip list

查找包

pip search "query statement"
